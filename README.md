# 🚀 CRUD Product Management System

An advanced, interactive Product Management System built from scratch using **Vanilla JavaScript, HTML5, and CSS3**. This system focuses heavily on delivering a premium User Experience (UX), data persistence, and performance.

## ✨ Key Features
* 🔄 **Full CRUD Operations:** Seamlessly Create, Read, Update, and Delete products with persistent data storage via `localStorage`.
* 📊 **Real-time Price Engine:** Instant live updates and calculations for net total prices including taxes, ads, and discounts.
* 🔍 **Live Smart Search:** Real-time dual-filtering system (Search by Product Name or Category) with non-matching fallback states.
* 🛡️ **Form Validation & Alerts:** Custom smart toast messages and notifications handling edge cases and empty fields.
* 📱 **Completely Responsive Layout:** Customized layout adjusting dynamically for Desktop, Tablet, and Mobile devices (using advanced CSS Media Queries and flexible table scrolling).
* 🎨 **Premium UI/UX:** Futuristic dark theme interface combined with a live background video background for a sleek experience.

## 🛠️ Technologies Used
* **Frontend:** HTML5, CSS3 (Flexbox, Grid, Media Queries, Custom `@keyframes` Animations)
* **Logic:** Vanilla JavaScript (ES6+, DOM Manipulation, LocalStorage APIs, Array Filters)

## 💻 Clean Code Approach
The codebase is strictly organized into clean, reusable functions, heavily documented with comments, and follows best practices for semantic structure and layout architecture.
